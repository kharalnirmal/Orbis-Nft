# Vibe Coded

VIBE CODED WEBSITE

Anyone can use it for their project — just give credit if you want.

Vibe Coded website — a dark, space-themed NFT landing built with React and Tailwind CSS.

Features:

- Dark space aesthetic with cinematic video backgrounds
- Liquid-glass cards, neon typography, and texture overlays
- Radix-based FAQ accordion and responsive layout

Built with: Vite, React, TypeScript, Tailwind CSS

Quick start:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
